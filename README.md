## 命令
* 查看版本号
  > 查看python版本：`python --version`  
  > 查看所有本地库：`pip list`  
  > 查看指定库版本：`pip show ${module_version}`  
* 安装指定版本库：`pip install ${module_name}==${module_version}`
* 更新到最新版本：`pip install --upgrade ${module_name}`
